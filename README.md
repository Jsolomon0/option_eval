# Option Eval

Next.js App Router project with a multi-leg options strategy builder and Tradier Sandbox chain integration.

## Local run

1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure environment:
   - Copy `.env.example` to `.env.local` and set `TRADIER_SANDBOX_TOKEN`.
3. Start dev server:
   ```bash
   npm run dev
   ```
4. Open `http://localhost:3000/options/strategy`.

## Tests

```bash
npm test
```

## Build

```bash
npm run build
```
